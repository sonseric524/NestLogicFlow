# NestLogicFlow - Dependency-Aware Data Generator

NestLogicFlow allows you to define interlinked field groups and generate consistent, structured mock datasets.

## Features
- Conditional field linking  
- CSV / JSON / SQL support  
- CLI-based control

## Usage
```bash
git clone https://github.com/your-username/NestLogicFlow.git
cd NestLogicFlow
python generate.py
